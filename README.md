# Week 1 - Retail Sales Excel Analysis

## Project Overview

This project was completed as part of my Data Analysis Internship at LogicStack.

The purpose of this project was to learn the fundamentals of data cleaning, formatting, analysis, sorting, filtering, and visualization using Microsoft Excel. I worked with a retail sales dataset and performed various analytical tasks to understand the data and generate meaningful insights.

---

## Objectives

- Understand the structure of a real-world retail sales dataset.
- Clean and format the dataset in Microsoft Excel.
- Perform data quality checks.
- Apply basic Excel formulas for analysis.
- Sort and filter data to answer business-related questions.
- Create summary tables and charts.
- Write observations based on the analysis.
- Learn how to upload and document a project on GitHub.

---

## Dataset Information

The dataset contains retail sales transaction records with the following information:

- Transaction ID
- Date
- Customer ID
- Gender
- Age
- Product Category
- Quantity
- Price per Unit
- Total Amount

---

## Tasks Completed

### 1. Dataset Understanding
- Explored the dataset.
- Identified the number of rows and columns.
- Determined data types.
- Identified unique product categories.
- Found the minimum and maximum customer age.

### 2. Data Cleaning and Formatting
- Created a cleaned copy of the dataset.
- Converted the data into an Excel Table.
- Formatted dates and numeric values.
- Adjusted column widths.
- Formatted headers.
- Applied Freeze Panes.

### 3. Data Quality Check
- Checked for blank cells.
- Checked for duplicate records.
- Verified Quantity values.
- Verified Price per Unit values.
- Calculated the Total Amount using formulas.
- Compared calculated values with the original Total Amount.

### 4. Basic Data Analysis
Performed calculations using Excel formulas:

- Total Sales Amount
- Average Sales Amount
- Minimum Sales Amount
- Maximum Sales Amount
- Total Quantity Sold
- Average Customer Age
- Youngest Customer Age
- Oldest Customer Age
- Total Number of Transactions
- Total Number of Unique Product Categories

### 5. Sorting and Filtering
Performed various sorting and filtering operations, including:

- Highest Total Amount
- Lowest Total Amount
- Clothing transactions
- Female customer transactions
- Quantity greater than 2
- Sorting Total Amount (Highest to Lowest)
- Sorting Age (Youngest to Oldest)

### 6. Category-Based Analysis
Created summary tables for:

- Total Sales by Product Category
- Total Quantity Sold by Product Category
- Total Sales by Gender
- Total Sales by Age Group 

### 7. Charts
Created the following charts:

- Bar Chart – Total Sales by Product Category
- Column Chart – Quantity Sold by Product Category
- Pie Chart – Total Sales by Gender

### 8. Observations
Prepared observations based on the analysis and results obtained from the dataset.
